# Design System

## Design Principles
- **Hierarchy over decoration**: Visual weight must guide the eye to the primary action.
- **Whitespace before borders**: Use spacing to group elements before relying on explicit dividing lines.
- **Motion communicates intent**: Animations should be fast, subtle, and meaningful (not just for show).
- **Accessibility before aesthetics**: Contrast, keyboard navigation, and screen reader labels are non-negotiable.
- **Consistency before originality**: Standardize radius, spacing, and colors rather than inventing one-off styles.

Our design system is built on top of `shadcn/ui` and Tailwind CSS.
